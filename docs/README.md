## 🛠️ 脚本

### Docker 脚本

位于 `scripts/docker/` 目录：

- `docker-build-push.bat` - 构建和推送脚本（Windows）
- `docker-build-push.sh` - 构建和推送脚本（Linux/Mac）
- `docker-build.bat` - 构建脚本（Windows）
- `docker-build.sh` - 构建脚本（Linux/Mac）
- `build-push-aliyun.bat` - 推送到阿里云脚本
- `立即推送.bat` - 快速推送脚本
- `构建推送.bat` - 构建和推送脚本

### 构建脚本

位于 `scripts/build/` 目录：

- `一键构建.bat` - 一键构建脚本
- `quick-build.bat` - 快速构建脚本
- `立即构建.bat` - 立即构建脚本
